Workspace Shortcuts

Fast jumps and common actions
  • Edit products here, then click Save Products.
  • Use the top tabs to switch between Catalog, Preview, Studio, and Sales without long scrolling.
  • Save or overwrite products-data.js in this project when prompted.
  • Commit and push your updated project to GitHub so Vercel redeploys the site.
  • The storefront reads products-data.js directly.
  • Use Mark as sold out to add the storefront sold-out design without deleting the product.
  • Keep product rows focused on image, name, price, material, story, specs, and gallery.
  • Use the Choose Image button to pick a local file and fill the image path.
  • Fill in material, story, and specs to power filters and the detail modal.
