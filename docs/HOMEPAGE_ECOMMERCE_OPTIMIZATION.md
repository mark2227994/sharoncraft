## Visual Improvements

### **Add Visual Hierarchy:**
- Use larger headings for primary sections
- Add subtle background colors (alternating cream/white)
- Increase spacing between sections
- Use more prominent CTAs (buttons, not just links)

### **Improve Mobile Experience:**
- Stack cards vertically
- Make CTAs full-width and touchable
- Reduce section padding on small screens
- Horizontal scroll for carousels

### **Enhance Micro-Interactions:**
- Product cards: hover shows quick view
- Category cards: slight lift on hover
- Testimonials: auto-rotate on desktop
- Stats: count-up animation on scroll

### **Add Missing Copy:**
- Category cards need descriptions
- Trust items need elaboration
- CTAs need more compelling wording
