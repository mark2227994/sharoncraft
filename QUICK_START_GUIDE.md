## 📦 Your Admin Dashboard Features

### 1. **Products** - Manage Your Inventory
```
📍 Location: /admin/products
What you can do:
  ✓ View all products (visible + hidden)
  ✓ Search by product name
  ✓ Filter by category
  ✓ Edit product details (name, price, stock)
  ✓ Toggle visibility ON/OFF (shows/hides from website)
  ✓ Delete products

Quick Start:
  1. Click "Add Product"
  2. Fill in name, price, category, stock
  3. Click "Create" → Product is live!
```

### 2. **Orders** - Track Customer Orders
```
📍 Location: /admin/orders
What you can do:
  ✓ See all customer orders
  ✓ Filter by status (pending, processing, shipped, delivered)
  ✓ Update order status (drag through workflow)
  ✓ View customer details
  ✓ Add tracking numbers

Quick Start:
  1. Click on an order
  2. Change status dropdown
  3. Add tracking number if shipped
  4. Order automatically updates
```

### 3. **Customers** - Manage Customers
```
📍 Location: /admin/customers
What you can do:
  ✓ See all customers who've ordered
  ✓ Search customer name, email, phone
  ✓ View total orders and amount spent
  ✓ Delete customer records

Quick Start:
  1. Type in search box (finds by name, email, or phone)
  2. View customer history
  3. Delete if needed
```

### 4. **Inventory** - Stock Management
```
📍 Location: /admin/inventory
What you can do:
  ✓ View current stock levels for all products
  ✓ See low-stock warnings (when below alert level)
  ✓ Update stock quantities
  ✓ Quick "+1" button for quick increments

Quick Start:
  1. Check "Show low stock only" to see items that need restocking
  2. Click a number to edit directly

```

### 5. **Categories** - Organize Products
```
📍 Location: /admin/categories
What you can do:
  ✓ Create new categories (Necklaces, Bracelets, etc.)
  ✓ Add subcategories to each

### 6. **Media** - Homepage Images

### 7. **Discounts** - Create Promo Codes
```
📍 Location: /admin/discounts
What you can do:
  ✓ Generate discount codes (SUMMER20, SAVE500, etc.)

  6. Set expiry date (optional)

## 🚀 Common Tasks

### Add a New Product
```
1. Go to Products
2. Click "Add Product" (top right)
3. Fill in:
   - Name: "Beaded Bracelet"
   - Description: Details about the piece
   - Price: 1500
   - Category: Choose from dropdown
   - Stock: 10
   - Check "Visible on Store" ✓
   - Check "New Arrival" ✓
4. Click "Create"
5. Product appears on website immediately!
```

### Create a Sale
```
1. Go to Discounts
2. Click "Create Discount Code"
3. Enter code: EASTER20
4. Type: Percentage
5. Value: 20
6. Max uses: Leave blank (unlimited) or set number
7. Expiry: Pick a date (e.g., Easter Sunday)
8. Click "Create Code"
9. Share code with customers: "Use code EASTER20 for 20% off!"
```

### Update Order Status
```
1. Go to Orders
2. Find customer order
3. Click status dropdown
4. Change: pending → processing → shipped → delivered
5. If shipped, add tracking number
6. Changes save automatically!
```

### Hide a Product from Website
```
1. Go to Products
2. Find product
3. Click "ON" button (green)
4. It turns "OFF" (red)
5. Product is immediately hidden from website
```

### Add Hero Slide for Homepage
```
1. Go to Media
2. Click "Add Slide"
3. Headline: "New Collection Available"
4. Subtitle: "Limited edition pieces"
5. Image URL: Paste image link
6. Click "Save"
7. Slide appears on homepage slideshow!
```

## ❓ Troubleshooting

### "Can't login"
- Check your email and password
- Make sure Caps Lock is OFF
- Try refreshing page
- Contact support if still stuck

### "Can't see products I added"
- Make sure "Visible on Store" is checked ✓
- Check the category exists
- Try refreshing page

### "Mobile is slow"
- Use Wi-Fi instead of mobile data
- Close other apps
- Try landscape mode
- Refresh the page

### "Changes not showing on website"
- Wait 10 seconds (takes time to sync)
- Refresh the website in another tab
- Check product is set to "Visible"
