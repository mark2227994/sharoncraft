Recommended Arrangement

Build one product in a calm step-by-step flow.

Start with the product identity, choose the best photo, write the customer-facing words, then open advanced details only if you want profit and stock tracking.

1. Start simple

Name, category, price, and badge are enough to begin a clean product draft.

2. Think visually

3. Save with confidence

Use the live preview and final checklist before pressing save.

Step 1
Basic product details

Start with the first things a customer notices and remembers.

The helper gives you a starter name from the category, description, details, and badge. You can always edit the result.

Pick where this product belongs. This helps customers find it in the shop.

Step 2
Photos and gallery

One clear main image is enough to start. Add more photos only when they help a customer decide faster.

Select a file to save it inside this browser for previews and product cards, or type an `assets/images/...` path if the file already exists in the project.

Simple photo flow
  1. Choose one clear photo first.
  2. Wait for the preview image to appear.
  3. Add more photos only if they show a new angle, color, or detail.
  4. Use `Selected gallery images` to choose the first image customers will see.

Search the saved product photos here and send them straight into the draft without leaving this step.

Step 3
Description and visibility

Keep the words clear and friendly. Say what it is, why it stands out, and where someone can wear or use it.

Easy formula: what it is + standout color or style + where someone can wear or use it + why it feels special.

Color or pattern Handmade detail Gift idea Wedding or daily use Comfort or size feel

Optional advanced details

Money and stock

Step 5
Preview and save

Read the product once like a customer, then save it with confidence.

Before you save
  1. Check the product name, price, and category.
  2. Make sure one clear image is selected.
  3. Read the short description once like a customer.
  4. Use the phone preview after saving if the item is important.
