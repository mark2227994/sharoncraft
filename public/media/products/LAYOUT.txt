=========================================

DROP YOUR PHOTOS IN THIS FOLDER: public/media/products/

NAMING RULE: [product-id]-main.webp  (main photo)
             [product-id]-2.webp     (second photo)
             [product-id]-3.webp     (third photo, optional)

PHOTO SIZE GUIDE
================
- Best size: 1000 × 1300 px (portrait 4:5 ratio) or 1200 × 1200 px square
- Minimum width: 900 px
- Keep each file under 300 KB if possible
- Format: WebP (preferred) or JPG


ADDING A BRAND NEW PRODUCT
===========================
1. Add photos here:  public/media/products/my-new-product-main.webp
2. Open this file:   data/store/products.json
3. Add a new entry with "id": "my-new-product" and images: ["/media/products/my-new-product-main.webp"]
4. Commit and push to GitHub → Vercel auto-deploys
