SHARONCRAFT PRODUCT PHOTO LAYOUT
================================

This folder now has category-first structure so new products are easier to add and easier to find later.

Keep existing flat files if they are already in use. Do not break old paths just to reorganize them.
Use the new folder structure for all new products going forward.


FOLDER STRUCTURE
================

Jewellery
  public/media/products/jewellery/necklaces/<product-slug>/
  public/media/products/jewellery/bracelets/<product-slug>/
  public/media/products/jewellery/earrings/<product-slug>/

Other categories
  public/media/products/home-decor/<product-slug>/
  public/media/products/gift-sets/<product-slug>/
  public/media/products/accessories/<product-slug>/
  public/media/products/bridal-occasion/<product-slug>/


SHOT ORDER
==========

Jewellery
  01-close-up
  02-worn-or-styled
  03-detail

Home decor, gift sets, accessories, bridal occasion
  01-hero
  02-in-use-or-styled
  03-detail


PHOTO RULES
===========

Necklaces, bracelets, earrings
  1. Start with one clean close-up.
  2. Add one worn-on-body photo when possible.
  3. Add one detail shot showing bead texture, clasp, or hand-finishing.

All other categories
  1. Start with one clear hero image.
  2. Add one styled-in-space or in-use image.
  3. Add one detail shot.


NAMING RULE
===========

Use the product slug in every exported image name.

Examples
  sunburst-cascade-necklace-main.webp
  sunburst-cascade-necklace-worn.webp
  sunburst-cascade-necklace-detail.webp

  kijani-mirror-duo-main.webp
  kijani-mirror-duo-styled.webp
  kijani-mirror-duo-detail.webp


ADMIN WORKFLOW
==============

The admin product form now shows a suggested project folder and uploads to a matching organized storage path.

Recommended flow:
  1. Drop raw photos into the matching category folder in this project.
  2. Pick the best close-up, styled, and detail images.
  3. Upload them through the admin form in that same order.
  4. Keep the product slug, folder name, and image names aligned.


ONE EXTRA RECOMMENDATION
========================

For every new jewellery product, keep one simple matching note in the product description or admin notes:
  "Pairs with drop earrings"
  "Pairs with a layered necklace"

That makes future bundles and "Wear it with" pairings easier to maintain.
