# SharonCraft Mini

Small Expo starter for a Play Store side project based on the SharonCraft brand.

## What is included

- warm, branded mobile home screen
- browseable handcrafted catalog
- product detail view
- persistent favorites using AsyncStorage
- WhatsApp and phone order actions

## Run locally

1. Open `mobile-app`
2. Run `npm start`
3. Scan the Expo QR code with Expo Go
