# Quick Guide for Your Mom - Admin Dashboard

## Welcome! 👋

Your son built an admin dashboard to help manage the online business. Here's how to use it.

---

## 🎯 Main Tasks

### **1. View New Orders**
- Go to `/admin/orders`
- See all customer requests
- Note which products they want
- Tell your son the timeline for each design

### **2. Update Product Status**
- When you finish a design, mark it as "ready"
- When shipped, mark as "shipped"
- This updates the customer automatically

### **3. Check Your Performance**
- Go to `/admin/leaderboard`
- See which designs sell best
- Focus on those popular items

### **4. View Revenue**
- Go to `/admin/analytics`
- See daily/weekly sales
- See total revenue

### **5. Add New Products**
- Go to `/admin/products/new`
- Upload photo
- Write description
- Set price
- Save
- Your son will handle the rest
