# Manual Smoke Checklist

Run these checks after major frontend changes or a deploy.

## Core pages

- Open `index.html`
- Open `shop.html`
- Open `product.html`
- Open `account.html`
- Open `contact.html`
- Open `privacy.html`
- Open `404.html`

## Shared UI

- Header renders correctly
- Footer renders correctly
- Product cards load without broken images
- Wishlist button still works
- Cart drawer opens and closes smoothly
- Cookie banner appears on first visit and hides after a choice

## Mobile checks

- Homepage scroll feels smooth
- Shop filters open and close correctly
- Product page buttons stay responsive
- No obvious text flash from old content to new content

## Commerce checks

- WhatsApp links open with the expected message
- Product prices render correctly
- Related products show on the product page
- Order tracking page still loads
